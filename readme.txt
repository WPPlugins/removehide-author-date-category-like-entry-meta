== Description ==

**Remove total entry meta or Specific meta:**

* Just install this plugin and it removes all entry meta.
* If you want to remove selected meta go to this plugin setting and select check boxes which meta want to hide.
* By default this plugin removes the total entry meta

== Installation ==

This section describes how to install the plugin and get it working.

In WordPress:

* Go to Plugins > Add New > search for :: Remove/hide Author, Date, Category Like Entry-Meta ::
* Press Install Now for the Remove/hide Author, Date, Category Like Entry-Meta plugin
* Press Activate Plugin

To install manually instead:

* Upload the remove-author-date-and-category-entrymeta  directory to the /wp-content/plugins/ directory
* Activate the plugin through the Plugins menu in WordPress
